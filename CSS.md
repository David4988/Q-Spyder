## CSS Syntax

### Selector

#### * the part of the CSS that selects the HTML elements **you want to style.**

```css
selector {
    propertyName: value;
    .
    .
    .
}
```

## 3 ways to apply CSS

### 1. Inline CSS

#### * CSS is applied to the **HTML elements directly.**

```css
<p style="color: red;">This is a paragraph.</p>
```
